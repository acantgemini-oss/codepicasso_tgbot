# 🎨 Code Picasso (Telegram Bot)

`Code Picasso` is an automated, zero-cost Telegram bot that transforms ugly, raw code blocks shared in chats into beautifully syntax-highlighted carbon-copy code images. It operates completely locally without external paid APIs using headless rendering.
Access through Telegram : @CodePicasso_bot

---

## ✨ Features
* **Zero API Costs:** Runs completely locally using Python libraries.
* **Smart Auto-Detection:** Automatically guesses the programming language of incoming snippets.
* **Beautiful Presentations:** Renders snippets inside an elegant dark-mode canvas with shadow profiling.
* **Mobile Friendly:** Converts low-readability mono text blocks into highly visible images optimized for small screens.

---

## 🛠️ Built With
* [Python 3.10+](https://www.python.org/)
* [python-telegram-bot](https://github.com/python-telegram-bot/python-telegram-bot) - Bot API framework
* [Pygments](https://pygments.org/) - Code syntax tokens compiler
* [Playwright](https://playwright.dev/python/) - Headless browser engine for image snapshots
